# E-Learning Platform

## Overview
The **E-Learning Platform** is a full-stack web application built using **Java, Spring Boot, JSP, and MySQL**.  
It allows users to manage students, courses, assignments, and track student progress.  

This project demonstrates **full-stack development skills**, including backend services, database integration, and JSP-based frontend views.

---

## Features
- **Student Management:** Add and list students with name and email.  
- **Course Management:** Create and view courses.  
- **Assignment Management:** Create assignments and associate them with courses.  
- **Progress Tracking:** View assignments per course.

---

## Tech Stack
- **Backend:** Java 17, Spring Boot, Spring MVC, Spring Data JPA  
- **Frontend:** JSP, HTML, CSS  
- **Database:** MySQL  
- **Build Tool:** Maven
